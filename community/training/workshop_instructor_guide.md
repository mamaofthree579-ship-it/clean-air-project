Workshop Instructor Guide

This guide helps teams teach community members how to build monitoring nodes.

---

## Workshop Structure (2 hours)

1. Introduction to Air Quality  
2. Hardware Overview  
3. Hands-On Node Building  
4. Firmware Flashing  
5. Dashboard Setup  
6. Q&A Session  

---

## Learning Goals

- Understand PM, VOCs, humidity  
- Learn to assemble a node  
- Learn to read values from serial monitor  
- Understand how open community data works  

---

## Materials Needed

- 1 Node kit per participant  
- Laptops w/ USB  
- Screwdrivers  
- Wi-Fi access
